# [MusEduLab](https://fkondela.github.io/MusEduLab/)
MusEduLab is an interactive ear-training lab for music students. Practise intervals, triads, and dominant chords, tailor tempo and instrument, generate rhythm or melody dictations with auto-checking, and switch light/dark plus Polish/English modes—everything to improve musical skills.

## Features
- Interval, chord, scale and dominant practice modes with tempo control
- Dictation generator with custom meter, tempo, key and instrument
- Language toggle (`PL`/`ENG`) and light/dark theme switcher
- Liquid-glass UI with animated accents

## Getting Started
- Visit the live site at https://fkondela.github.io/MusEduLab/ and start training
